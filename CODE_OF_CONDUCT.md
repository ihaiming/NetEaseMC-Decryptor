# 贡献者行为准则

## 我们的承诺

为了营造一个开放、友好、包容的社区环境，我们作为贡献者和维护者承诺：无论年龄、体型、残疾、种族、性别特征、性别认同、经验水平、教育背景、社会经济地位、国籍、个人外貌、宗教或性取向如何，所有人都能在我们的项目中受到尊重和友好对待。

## 我们的标准

**有助于创造积极环境的行为包括：**

- 使用友好、包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注社区整体利益
- 对其他社区成员表达同理心

**不可接受的行为包括：**

- 使用带有性暗示的语言或图像，以及任何形式的性骚扰
- 挑衅、侮辱或贬低性评论，人身攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息（如地址、电子邮件）
- 其他在专业环境中被合理认为不恰当的行为

## 责任与执行

项目维护者有责任明确可接受行为的标准，并对任何不可接受的行为采取适当且公平的纠正措施。

如果遇到违反本行为准则的情况，请通过以下方式联系我们：

- **GitHub Issues**：[点击提交](https://github.com/ihaiming/NetEaseMC-Decryptor/issues)

我们将认真对待每一份举报，并承诺保密举报人身份。在调查后，维护者将根据实际情况采取相应措施，包括但不限于：

- 警告违规者
- 要求公开道歉
- 删除或编辑不当内容
- 暂时或永久封禁违规贡献者

## 适用范围

本行为准则适用于所有项目空间（包括 Issues、Pull Requests、讨论区等），也适用于个人在公共场合代表项目或其社区时。
